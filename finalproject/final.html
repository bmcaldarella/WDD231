Peruvian Flavors — Site Plan

Individual project (3 pages) — Pure HTML, CSS, and JavaScript. This document defines the name, purpose, scenarios, colors, typography, and wireframes of the homepage.

Site Name

Peruvian Flavors

A short, clear, and memorable name that directly communicates the website’s purpose: Peruvian recipes and food culture.

Domain:peruvianflavours.com.

Site Purpose

This three–page website is designed to help users discover, learn, and save authentic Peruvian recipes. The Recipes page will showcase at least 15 dishes with details like name, cooking time, difficulty, and category, loaded from a local JSON file (with the option to expand using a public API). The Cookbook page will let users save their favorite recipes using localStorage. An accessible modal will also be included to display ingredients and step-by-step instructions.

Scenarios

  • “I want to learn how to make Ceviche, Ají de Gallina, and Lomo Saltado with clear steps and photos.”
  • “Where can I save my favorite recipes so I can find them quickly later?”
  • “How can I explore traditional Peruvian dishes and their cultural background?”
  • “Can I filter recipes by ingredients I already have at home?”
  • “I’d like to share a Peruvian recipe with my friends—what’s the easiest way?”

CSS

The final site will use valid CSS with no unnecessary duplicates. Visual consistency will be maintained (proximity, alignment, repetition, contrast, and spacing). Responsive navigation: hamburger menu on mobile, horizontal navigation on desktop. Layout will avoid horizontal scrolling from 320 px and up. Page weight < 500 kB.
